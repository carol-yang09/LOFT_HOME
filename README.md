# LOFT_HOME 旅館訂房

[![cover1.png](https://i.postimg.cc/ZK8hSntS/cover1.png)](https://postimg.cc/Czx2CFxP)

## Demo

https://carol-yang09.github.io/LOFT_HOME/dist

## 簡介

使用 Vue Cli 2 、 Vuex 及 Vue Router 旅館訂房網站，主要特色有

* 串接訂房 API
* 搜尋可預約房型
* 訂房結帳
* 圖片輪播
* 房型列表展示
* 單一房型詳細介紹

## 技術
* Vue Cli 2
* Vuex
* Vue Router
* Vue Component
* 串接 API
* ESlint
* scss手刻
